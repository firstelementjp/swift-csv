## Manual Installation

1. [Download latest version](https://github.com/firstelementjp/swift-csv/releases/latest)
2. Extract the downloaded ZIP file
3. Upload `swift-csv` folder to `/wp-content/plugins/`
4. Activate the plugin from WordPress admin

## Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
- Memory limit: 64MB or higher (for large CSV processing)

## Initial Setup

After plugin activation, configure basic settings from the "Swift CSV" menu in the admin panel.
